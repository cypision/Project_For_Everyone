branch 종류 구분하기

일반적으로 분류하면, 4가지로 나눌수 있다.


참고자료 원본 링크

1.Local Branch
1.1 Non-tracking local branches
어떠한 branch 랑도 연관되어 있지 않은 local repo의 branch
1.2 Tracking local branches
주로 remote-tracking branch 랑 연결되어 있는 local repo의 branch. " git branch --track " 명령어로 확인 가능
1.3 remote tracking branches
local 에 존재하는 remote branch의 복사본이다 " git branch -avv " 명령어로 확인가능

2.Remote Branch
2.1 remote branch
물리적으로 원격 reposittory에 존재하는 branch 를 말한다. 지금까지 얘기했던, 부분을 그림으로 보면 좀더 이해가 쉽다
흰 글씨들이 tracking local branch이며, 파랑색깔이 remote-tracking-branch 이다. 빨강색으로 된 branch들이 remote branch 이다
